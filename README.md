# Medium Clone

A responsive clone of the Medium website created using HTML, CSS, and Vanilla JavaScript. This project focuses on recreating the design, layout, and user interface of Medium for learning and practice purposes.

## Features
- Responsive navigation bar
- Hero section
- Article feed layout
- Sidebar navigation
- Clean and modern UI inspired by Medium
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Purpose
This project was built for educational purposes to improve frontend development skills by replicating a real-world website design.

## Disclaimer
This is a personal educational project and is not affiliated with or endorsed by Medium.
